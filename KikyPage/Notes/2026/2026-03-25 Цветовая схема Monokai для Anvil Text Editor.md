Еще одна цветовая схема для редактора Anvil, на этот раз вдохновленная темой Monokai, с которой я впервые познакомился в редакторе Sublime Text.
![[Pasted image 20260325083540.png]]

**Как установить тему:**
1. Скачайте файл `style.js` из [репозитория](https://github.com/KikyTokamuro/anvil-monokai).
2. Поместите его в директорию с конфигурацией Anvil:
    - **Linux/macOS:** `~/.anvil/`
    - **Windows:** `%USERPROFILE%\.anvil\`
3. Чтобы применить тему, можно:
    - Выполнить команду `LoadStyle`.
    - Или просто перезапустить редактор — тема подгрузится автоматически.
