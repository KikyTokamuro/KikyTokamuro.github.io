#dev #xcode #plan9 #acme #anvil 

Давно искал аналог легендарного редактора [Acme](https://9fans.github.io/plan9port/man/man1/acme.html), который бы стабильно работал вне экосистемы Plan 9. И вот, наконец, наткнулся на [Anvil](https://anvil-editor.net/) — редактор, который сохраняет философию Acme, но при этом чувствует себя как дома в современной среде.

![[Pasted image 20250823141905.png]]

**Ключевые отличия по сравнению с оригинальным Acme:**
- Подсветка синтаксиса для большего удобства.
- Удалённое редактирование файлов по SSH прямо из редактора.
- Поддержка нескольких курсоров для эффективного редактирования.
- Знакомые клавиатурные сочетания.
- Умный синтаксис для команд с пробелами, например: `◊search term◊`. Команду не нужно выделять целиком — достаточно кликнуть в любое место между символами-ромбами.
- И множество других приятных улучшений.

Кроме того, Anvil поддерживает цветовые схемы, которые описываются простым JSON-файлом.

Так что, я написал для него [простенькую тему](https://github.com/KikyTokamuro/anvil-xcode) вдохновленную цветами из Xcode:
![[Pasted image 20250823141832.png]]

**Как установить тему:**
1. Скачайте файл `style.js` из репозитория.
2. Поместите его в директорию с конфигурацией Anvil:
    - **Linux/macOS:** `~/.anvil/`
    - **Windows:** `%USERPROFILE%\.anvil\`
3. Чтобы применить тему, можно:
    - Выполнить команду `LoadStyle`.
    - Или просто перезапустить редактор — тема подгрузится автоматически.
